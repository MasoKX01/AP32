¿Cómo funciona el freno cognitivo?

El firmware del Arduino Mega monitorea en tiempo real el caudal mediante el flujómetro (Pin 3) y el tiempo de uso. Al superar cualquiera de los umbrales definidos, activa una respuesta escalonada de alerta visual que replicas ahora mismo en este simulador.

Los LEDs virtuales responden exactamente como el módulo LED real (Pin 6) del prototipo físico: verde durante la operación normal, rojo estroboscópico en estado de excedente.

  • 0 – 10 SEGUNDOS Operación normal. LEDs verdes. Consumo esperado ≤ 580 ml. El vórtice recircula el agua de enjuague sin consumo de red.
  • 10 SEGUNDOS / 580 ML ⚠ Umbral superado. Activación del freno cognitivo: LEDs cambian a rojo estroboscópico, el display LCD muestra alerta. El sistema fuerza la conciencia del usuario.
  • 15 SEGUNDOS Bloqueo antivandálico automático. La electroválvula del circuito primario se cierra. Solo el circuito secundario (vórtice) permanece activo.

Arquitectura Mecatrónica

Mapeo de pines del microcontrolador Arduino Mega 2560 y descripción funcional de cada módulo.

Pin Componente Descripción Tipo Estado
D2 Sensor Infrarrojo Detección de presencia de usuario. Activa el ciclo automáticamente. INPUT DIGITAL ACTIVO
D3 Flujómetro YF-S201 Medición de caudal en tiempo real. Interrupciones por pulsos por litro. INPUT INTERRUPT ACTIVO
D6 Módulo LED RGB Indicador visual de estado. Verde = normal / Rojo = alerta / Estrobo = exceso. OUTPUT PWM ACTIVO
D7 Electroválvula Primaria Control del circuito de higiene. Cierre automático al activar bloqueo antivandálico. OUTPUT RELAY ACTIVO
D8 Bomba de Vórtice Activa el circuito secundario de recirculación en lazo cerrado (0 ml de red). OUTPUT RELAY ACTIVO
A0 Sensor de Nivel Monitoreo continuo del nivel del depósito de recirculación. INPUT ANALÓGICO ACTIVO
D9 Buzzer de Alerta Señal sonora en modo de excedente. Complementa el freno cognitivo visual. OUTPUT DIGITAL STANDBY
I2C Display LCD 16×2 Muestra tiempo, volumen y mensajes de estado en protocolo I²C. I2C BUS ACTIVO
